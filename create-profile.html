Crear Perfil QA Manualmente

Abre la consola del navegador (F12) y ejecuta esto:

const profile = {
    id: 'profile-manual',
    name: 'Tu Nombre',
    email: 'tu@email.com',
    resume: 'QA Engineer con experiencia en testing',
    skills: [
        { name: 'Manual Testing', years: 3, level: 'intermediate', category: 'testing' },
        { name: 'Selenium', years: 2, level: 'intermediate', category: 'automation' },
        { name: 'Python', years: 2, level: 'intermediate', category: 'programming' },
        { name: 'API Testing', years: 2, level: 'intermediate', category: 'testing' },
        { name: 'Agile', years: 3, level: 'intermediate', category: 'methodology' }
    ],
    totalExperience: 3,
    availability: 'immediate',
    location: 'Venezuela / Latin America',
    preferredLocations: ['Remote', 'Venezuela', 'Latin America'],
    createdAt: new Date(),
    updatedAt: new Date()
};

localStorage.setItem('qa_profile', JSON.stringify(profile));
alert('✅ Perfil creado! Recarga la página.');

Instrucciones

  1. Abre http://localhost:3000
  2. Abre la consola (F12)
  3. Copia y pega el código de arriba
  4. Presiona Enter
  5. Recarga la página
  6. Ve a "Job Search" y busca trabajos
